FICHIERS PRÉPARÉS DANS : github_upload\

══════════════════════════════════════════════════════════════════
  ÉTAPE 1 : CRÉER LE REPOSITORY GITHUB
══════════════════════════════════════════════════════════════════

1. Allez sur : https://github.com
2. Connectez-vous (ou créez un compte)
3. Cliquez sur le bouton vert "New" (en haut à gauche)
4. Nom du repository : tsubasa-dashboard
5. Description : Trading Performance Dashboard
6. Visibilité : Public
7. NE PAS cocher "Add a README file"
8. Cliquez : "Create repository"

══════════════════════════════════════════════════════════════════
  ÉTAPE 2 : UPLOADER LES FICHIERS
══════════════════════════════════════════════════════════════════

1. Sur la page du repository, cliquez : "uploading an existing file"
   Ou : "Add file" -> "Upload files"

2. Glissez-déposez les 5 fichiers du dossier github_upload\ :
   - requirements.txt
   - tsubasa_dashboard.py
   - trade_history.json
   - README.md
   - .gitignore

3. Message de commit (en bas) :
   "Initial dashboard setup with data"

4. Cliquez : "Commit changes"

══════════════════════════════════════════════════════════════════
  ÉTAPE 3 : DÉPLOYER SUR STREAMLIT CLOUD
══════════════════════════════════════════════════════════════════

1. Allez sur : https://share.streamlit.io

2. Connectez-vous avec GitHub
   - Cliquez : "Sign in with GitHub"
   - Autorisez Streamlit Cloud

3. Créez une nouvelle app
   - Cliquez : "New app"

4. Configuration :
   Repository : VOTRE_NOM/tsubasa-dashboard
   Branch : main (ou master)
   Main file path : tsubasa_dashboard.py

5. Cliquez : "Deploy!"

6. Attendez 1-3 minutes
   - Streamlit installe les dépendances
   - Logs en temps réel

══════════════════════════════════════════════════════════════════
  RÉSULTAT : DASHBOARD EN LIGNE !
══════════════════════════════════════════════════════════════════

Votre dashboard sera accessible à :
https://votre-app.streamlit.app

══════════════════════════════════════════════════════════════════
  POUR METTRE À JOUR PLUS TARD
══════════════════════════════════════════════════════════════════

Option 1 - Upload manuel (simple) :
1. GitHub -> trade_history.json -> Icône crayon (Edit)
2. Remplacer le contenu
3. Commit changes
4. Streamlit se met à jour automatiquement (~30s)

Option 2 - Installer Git (pour automatisation) :
1. https://git-scm.com/download/win
2. Installer Git
3. Relancer setup_git_dashboard.bat
4. Mise à jour automatique activée
